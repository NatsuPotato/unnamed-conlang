内容語

Content words are written in hiragana.

機能語も接辞も

Function words and affixes are written in kanji.

トキポナと同じように、単語は文脈によって名詞にも動詞にもできる。

| Onset | Character |
|-------|-----------|
| *     | あえいおう |
| p     | はへひほふ |
| t     | たてちとつ |
| k     | かけきこく |
| m     | まめみもむ |
| n     | なねにのぬ |
| s     | させしそす |
| j     | やゑゐよゆ |

日本語の異音はありませんよ。例えば、「ち」の発音は\[ti]です、\[tɕi]ではない。

```
食す / eat
食せす / ate
```
