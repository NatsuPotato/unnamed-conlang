Vowels: i, a, u

## Noun inflection

Nouns are inflected for case.

| Case       | Affix  |
|------------|--------|
| Nominative | -Ø     |
| Oblique    | -na    |
| Vocative   | -ya    |

* The vocative entails the imperative, and can be put on the 2nd person pronoun

## Syntax

Strongly head-final

SOV

>Tami, eat the food!
>
>Támiya tárina váyu!

>Eat the food!
>
>Úya tárina váyu!
