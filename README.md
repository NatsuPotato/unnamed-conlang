# unnamed conlang

Shrimple

## Phonology

CV

Stress falls on the penult of the root

|         | Bilabial | Alveolar | Palatal | Velar |
|---------|----------|----------|---------|-------|
| Nasal   | m        | n        |         |       |
| Stop    | p        | t        | c       | k     |
| Fric.   |          | s        |         |       |
| Squish  | w        | r /ɾ/    | j       |       |

Vowels: i, a, u

## Noun inflection

Nouns are inflected for case.

| Case       | Affix  |
|------------|--------|
| Nominative | -Ø     |
| Oblique    | -na    |

## Verb inflection

done with particles

Progressive, "can," Imperative, etc
